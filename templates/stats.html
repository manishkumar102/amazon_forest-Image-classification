The Prevalent Issues

Every minute,the world loses an area of forest the size of 48 foootball fields.And deforestation in the Amazon Basin accounts for the largest share,contributing to reduced biodversity , habitat lost , climate change, and other devastating effects.But better data about the location of deforestation and human encroachment on forests can help governments and local stakeholders respond more quickly and effectively. Labelling satellite image chips with atmospheric conditions and various classes of land cover/land use will help in building algorithms that will help the global community to better understand where,how and why deforestation happens all over the world - and ultimately how to respond.

Project Goal:

This project is an attempt to apply data science and machine learning techniques on the satellite image chips to label them.

  1. To be able to predict if there is any deforestation happening based on the sattelite images
  2. Detect useful objects like water,road,selective logging, and more through satellite image using machine learning.
  3. Create a user interface satellite images can be uploaded.
  4. Maximize the F2 score for evaluation of the model.
  5. Handle the partial clouds and haze images to not affect the labelling.
